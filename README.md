# Space-invaders-console-
a simple space invaders but can be pasting into Google inspect console

A console-injected Space Invaders clone you can paste into browser DevTools.

## Run

1. Open any webpage.
2. Open DevTools Console.
3. Paste the contents of `space-invaders.js` and press Enter.

## Features

- Classic 5x11 invader formation with accelerating movement.
- Player ship with single-shot arcade-style firing.
- Enemy return fire from the bottom-most invaders.
- Destructible bunker shields.
- Mystery UFO flyby bonus points.
- Score, high score, lives, and wave progression.
- Draggable game window.
